# 如何让自进化产生「质变」· 研究综合与路径

> 4 路并行调研综合(开放式进化 / 涌现与相变 / 进化形式本身与 self-play / 计算创意)。
> 问题:固定 DeepSeek 模型 + 无客观奖励 + 主观创作,现有自进化只是收敛性优化、逼近模型天花板——**怎样产生真正的质变,而非渐进调优?**

## 0. 先立诚实的边界(决定一切)

**真·相变在 AI 里全部由"改权重/训练"驱动——固定模型 + 调提示参数,等于关掉了相变主引擎。**
- Grokking(长训后突然泛化)、LLM 涌现能力、in-context learning 的突现,机制都是**权重内部表示重组/电路成形**。固定 DeepSeek 无法触发这一类相变。**别指望"积累到某天模型突然会写得更好"——做不到。**(arXiv:2201.02177 / 2206.07682 / 2209.11895)
- 你最可能观测到的"突然质变"**多半是度量假象(Mirage)**:连续的小改进被一个 0/1 验收指标(过审率/达标占比)的非线性放大成"看似跃迁"。换连续指标看,曲线是平滑幂律。(Schaeffer, arXiv:2304.15004)
- 无 ground-truth 的纯自评自进化有硬上限:反复自反思**边际递减**、对强基座甚至变差;长期用 LLM 当奖励会 **reward-hack 后崩溃**(刷"Let's think step by step"拿高分、长度暴涨)。(arXiv:2210.11610 / Self-Rewarding 崩溃)

→ 所以本文的"质变"**不是**权重级能力跃迁(那需训练),而是:**①把固定模型推入它绝不会主动采样的状态空间 + ②改变"概念空间"本身(变革性创造) + ③用人类信号突破自动评委天花板 + ④用开放式机制对抗收敛**。

## 1. 理论钥匙:Boden / Wiggins —— 质变 = 改"概念空间",不是空间内搜得更好

这是整篇的承重墙。**Boden 创意三分**:combinational(重组) / **exploratory(在固定"概念空间"=一组使能约束内探索)** / **transformational(改变定义空间的底层约束/规则)**。
- 我们现在的自进化 = **exploratory**:在固定生成模板/叙事框架内磨参数、磨避雷。永远突破不了那个空间的边界。
- 质变 = **transformational** = 丢弃/改写一条使能约束(如无调性音乐丢掉"必须有主音"),让"旧空间里不可能的作品"成为可能。判据不是"更新颖的点",而是"**生成地形被重构**"。
- **Wiggins 的可操作桥梁(关键)**:transformational creativity ≈ 在 **meta 层**做的 exploratory creativity——"改规则"本身可被建模为"在规则空间里搜索"。→ 给出明确设计模式:**双层进化 = 内容层(写章节) + 规则层(写约束)**,让进化作用在规则层。
(Boden《Creativity in a nutshell》;Wiggins 2006《Creative Systems Framework》;ICCC 2019 扩展)

## 2. 开放式判据:最大化 novel ∧ learnable(收敛 = novel 归零)

《Open-Endedness is Essential for ASI》(Hughes/Clune 2024, arXiv:2406.04268)给出最干净的目标函数:对一个**观察者**,产出序列须**同时 novel(惊喜不封顶)且 learnable(看更多历史能更会预测=有规律的惊喜)**。
- 两反例钉死张力:雪花屏(可学但不新)、随机换台(新但不可学)。
- **你现在的病根 = learnable 拉满、novel 归零 = 收敛。** 解药:把目标从"最大化质量分"改成"最大化 novel ∧ learnable"。

## 3. 质变路径:五层架构(固定模型不动,变的是喂给它的空间定义)

### 层①(真突破)进化「约束/世界规则/叙事范式」作为一等公民 —— 改概念空间
- 借:Boden 使能约束 + Wiggins meta-rules + POET"环境-解协同进化" + AI-GAs 第三支柱"自动生成学习环境"。
- 落地:把世界设定/叙事范式/写作约束抽成一份**显式、可变异的 constraint spec**(如"魔法必须有代价""禁用全知视角""每章须有一次叙事失败""不可靠叙述者")。进化算子不只改文字,还**增删/改写其中 1-2 条约束 → 生成子世界 → 跑若干章看是否打开新可能**。Dramatron 的层级骨架(logline→角色→情节→场景→对白)可当这份 spec 的载体。
- 为何真突破:同一模型在**全新硬约束**下的输出分布,可达无约束时永不采样到的区域。这是结构性新桥段的真来源。配最小准则(下层④)与多世界 A/B 兜底防崩。

### 层②(造奖励)人造"可验证的局部 ground-truth" —— self-play 质变的内核
- 关键事实:AlphaZero / AutoML-Zero / Absolute Zero 的质变**全部依赖可验证奖励**(胜负/准确率/代码执行器)。创作没有全局胜负,但能**程序化校验子目标**:伏笔必回收(NER+指代,章N埋点须≤M章兑现)、悬念兑现率、信息增量(新实体/新关系计数,防原地踏步)、设定一致性(角色属性/时间线/世界观不自相矛盾)、契诃夫之枪(提及的关键道具必须用)。
- 落地:让模型像 Absolute Zero 那样**自提"挖坑→填坑"任务对**,用这些校验器当"执行器式"可验证奖励来筛选/驱动。
- **诚实红线**:可验证子目标 **只作底线门槛/课程信号**,不作最终目标——否则"机械回收伏笔但无美感"。文学质量仍交评委(且评委要防崩溃)。

### 层③(破天花板)「议事」人类在环 = 突破自动评委天花板的方向盘 —— 最关键
- 最强证据:**Picbreeder** 人类策展 74 代育出"骷髅";把"骷髅"设成自动目标,20次×3万代复现不出——**踏脚石不像终点,目标驱动(客观分)因欺骗性地形错过伟大,人类策展+新颖性反而到达。**
- 落地:自动评委(DeepSeek 品味/surprise)负责**海量初筛 + 维持多样性**;你已有的**「议事」**里作者**偶尔对少量精英选优/否决**,这稀疏人类信号专做机器做不好的两件事:(a) **批准/否决规则层的约束突变**(哪条"完美但出格"的新范式值得留——Wiggins 的 aberration 判定本质是人的);(b) 校准 AI 评委、纠其系统性审美偏差。**人管"方向/质变合法性",机器管"通量/多样性"。** 人负担极低(稀疏、只看精英)。

### 层④(防收敛)Novelty/QD 压力 + 最小准则保护垫脚石 + 进化多样性轴
- 借:Novelty Search(放弃目标只追新颖,欺骗性任务上胜过目标搜索)+ MCC 最小准则 + QDAIF/ELM(固定模型 + LLM 当变异算子+评委跑 MAP-Elites,无 ground-truth 创作域已验证)。
- 落地:① 录用要"够好 ∧ 足够远(与近K章语义距离) ∧ 有趣";高质量但太像近作的章节**反而拒收**(直接对冲已被实证的 LLM 创作同质化 mode collapse,arXiv:2501.19361)。② 对实验分支用**最小准则门(自洽∧不崩∧足够新即存档)而非贪婪择优**——否则"暂时更差但开新方向"的垫脚石全被杀死、系统必然收敛。③ **让 DeepSeek 定期提议新的多样性轴**(治 QDAIF"人手工轴、轴外不发散"的致命短板)。

### 层⑤(累积)分代文化累积 —— 质变常是跨代而非单次
- 借:Cultural Accumulation in RL(Oxford×DeepMind 2024):分代学习 + 平衡"社会学习 vs 独立探索",累积代 > 单一长生命。
- 落地:世界分代,**后代继承前代被议事批准的约束/范式,但强制每代保留独立探索配额 + 保留分叉/重置逃生口**(坏约束可剪枝)。

## 4. 硬天花板与反模式(必须时刻记住)

1. **固定模型 = 原始写作能力的硬上限**,prompt/param 进化爬不穿。真要抬升靠更强 LLM。
2. **看似"突然质变"先证伪**:同时用连续指标(评分均值/编辑距离/人评 Likert/困惑度)监控;只有阈值指标跳=假象。
3. **自评自奖会被 reward-hack 后崩溃**:评委必须与生成器解耦(不同 prompt/模型/温度)+ 人工锚定校验集 + 反代理指标(长度/复读/伏笔开了不收)。
4. **自动评委的"品味"本身是天花板**(ASAL/OMNI:FM 看不出的新颖它发现不了)——所以层③人类在环不可省。
5. **过度优化可验证子目标 → 机械无美感**:子目标只作门槛,不作目标。

## 5. 最小落地路径(建议先做这个原型验证)

把现有"内容层进化"上移一层,做一个**最小双层进化 + 议事闸门**原型:
1. 给每个世界加一份 `constraints.json`(3-5 条显式使能约束)。
2. 进化算子每隔若干卷,除了现有的参数/避雷,**额外提议改写1条约束 → 开一个子世界 A/B 跑 8 章**。
3. 自动评委按 novel∧learnable + 可验证子目标(伏笔回收率/信息增量/一致性)给子世界打分;**最小准则门**(不崩∧够新)即入存档。
4. **「议事」推给作者**:"子世界改了约束X,产出了这些——保留进主线 / 否决 / 再分叉?" 人决定规则层突变的去留。
5. 被批准的约束进入全局传承(分代累积),播种后续世界。
全程用连续指标证伪"是否真有质变 vs 假象"。core/ 不动,纯 app 层。

## 6. 一句话总纲

固定模型下,**质变不来自更好的参数,而来自:更陌生的约束(改概念空间)+ 人造可验证子目标(局部 ground-truth)+ 人类稀疏策展(破自动评委天花板)+ 开放式 novelty(对抗收敛)+ 跨代累积**。它能让系统从"探索性(空间内打磨)"走向"变革性(改变空间)"——这是固定模型下"质变"的真实、且有限的含义。真正无限的天花板抬升,仍要靠更强的底座模型。
