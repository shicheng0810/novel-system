## 0. 一句话结论（先定调）

降密度（commit 2caf8ae 的 M1 prompt「一段一事·宁短」+ M2 采样）与留白 amplify **方向完全正确，本报告不回退、不加码**。用户钦点的两个问题（信息密度=慢而满、详略不当=慢而拖）在当前在跑章里**已基本收敛**——琐碎动作不再铺三段七步。真正在劣化阅读的是**六路调研一致指向的、与降密度正交的另一簇问题**：它们不是「单段塞太多」（空间维，已治），而是「**段与段没快慢落差、章与章在复读、全员一个嗓子、收尾一个姿势、比喻一套模具、避雷账本空转**」（节奏维 / 跨章时间维 / 声口维 / 机制维）。降密度让每段变轻了，但**所有段一样轻、一样长、一样速、一样温**——补的就是这层起伏、多样与层次。

并且发现一个**隐藏的联动副作用**（用户已预判）：M1 的「一段一事·宁短」在**压低单段信息量**的同时，把「详略」一并压平了——过场和高光被同等地「写从容即收」，于是**没有该一笔带过的快段、也没有该驻足的慢段**，催生了「全章匀速·详略不分」。这条联动在 §二 详述。

## 一、可读性问题清单（按「伤阅读程度」排序）

### 【定位·已修】信息密度=慢而满（空间维·物象堆叠）— 已收敛 ✅
- 本质：单段堆叠多重微距静物/感官特写，读者一段内要处理太多物象。
- 现状：降密度 M1+M2 见效，单段物象总量已降。`warm-fitness.ts` W_breath（对话/千字）守住了「情报播报」边界（好章 3.7-5.9 vs 高密度 9.2-13.1）。**无需再加码**，否则伤及好章已有的从容细描。

### 【定位·已修但有副作用】详略不当=慢而拖（详略维）— 主体收敛，但催生了问题①
- 本质：把日常给予/位移动作做慢镜头逐帧分解。
- 残留实例（`yunyou/ch-0031` L101 慧明施饼）：「卸篓→摸袋→松袋口→抽两块→顿→抽第三块→兜布→走→蹲→搁」——一个「给饼」拆 10 个微动作。对照好章 `ch-0001` L43 师父给饼一笔带过：「从袖子里摸出个油纸包，递过去，『还有两块饼，拿着。』」
- 现状：频次已低（整章 1-2 处），**不构成主问题**。但其降密度修法（一段一事·宁短）**联动催生了下面的①节奏单一**（见 §二）。

### ① 节奏单一·全章匀速（详略维的真问题·正交于降密度）— 高
- **本质**：全章中速匀速，缺「一句话甩一段的快段（提速/留白）」与「驻足细描的慢段」之间的落差。降密度让段变轻，没让段「有快有慢」。

|---|---|---|---|---|---|
| 好 renjian-killed ch1 | 130 | **25.5** | 28.9 | **55** | 30 |
| 好 shanju ch3 | 116 | 38.0 | 24.3 | 27 | 24 |

- **实例·坏**：`yunyou/ch-0031` L93-121 慧明桥头连续 6 段全是 60-120 字等长描写块（竹笠/灰袍/篓子/干饼/妇人/孩子/念珠/水声逐一细描），**无一个单句快段切进来喘息**。
- **实例·好**：`ch-0001` L53-59 在井沿/斋堂细描之间反复插入单句快段提速——「他低头看路。」「师父没回头。」「虚谷便不再问了。」「一步一步，低头看路。」——像鼓点，给长段做留白和提速对照。
- **怎么影响阅读**：读者像看一条没有快慢的传送带（yunyou ch31 中位 50.5 字、快段只剩 12 个）。缺「停顿—铺展—再停顿」的呼吸，**满而匀**，读着累且单调。

### ④ 对白寡淡·全员同腔（声口维）— 中高
- **本质**：温润被过度执行成「全员单字短句、一个调门」。PENMANSHIP「对白重言外之意、一句别塞多条事实」(L41) 被过度执行成「对白尽量短、尽量不说话」——把「克制」做成了「失声」。
- **量化**（对白句中 ≤4 字短句占比）：yunyou ch31 **56%**（坐/明日/吃/将就穿）、renjian ch32 **48%**（那就去/会/是/认得）、shanju ch31 27%。
- **实例·主角可无损互换**：虚谷「嗯/我知道/记得/那就蒸」、云栖「嗯/我知道你的意思/还是那些/留步」、云归「嗯/能走/我背/走吧」——抽出台词读者认不出是谁。
- **实例·好**：`ch-0001` 师父「你到了就知道了」「凡心未歇」自带说教者笃定冷峭；挑夫「这天儿可不早了」「道长赶路呢」自带山野人絮叨热络——两种声音一听可辨。`shanju/ch-0003` 庄溪「不是看炉子，是冷」一句立人。
- **怎么影响阅读**：所有角色被磨成同一副「淡而稳」嗓子；情感落点也趋同（章末都「不应/不答/沉默」）。寡淡压过留白，读者拿不到人物层次。好的温润对白应有**人话密度差**：主角惜字，但来客可絮叨、可寒暄、可说半截话。

### ⑤ 章末收尾套路化·静物渐息（结构维·软约束已下达但未遵守）— 中高
- **本质**：收尾高度集中在「一个静物 + 一个渐息/渐淡/归位的微动作」（茶烟久久不散、水渍洇开又淡去、灯一直没熄）。
- **量化**（各章末段分类「静物渐息」占比，越跑越严重 = 自进化漂移）：

| 在跑 renjian | 44% |

- **实例**：shanju ch29「茶烟…久久不散」/ch31「一层覆一层地洇开，又一层覆一层地淡去」；yunyou ch28「像灶台上那盏一直没熄的灯」；renjian ch29「门轴涩涩地叫了一声，又收住了」。
- **关键**：`longrun.ts:203` secPrompt 末段**已写**「余味不要总落在静物停寂或一声渐息上……换着来，别每章都收在同一种静止里」——但 shanju 仍 66%。**软指令已下达但未被遵守**，证明软约束不够、需升级为机检。

### ⑥ 避雷账本（avoid）品类串台空转（机制维·根因层·最高杠杆）— 高（P0 机制）
- **本质**：本应是去套路主力机制的 `evolution.json:avoid`，在三个温润世界里**全是爽文/现代题材的幽灵条目**，与温润正文零交集——避雷机制在防一批温润世界**根本不会写**的爽文喻。
- **量化铁证**（本次复核三世界 avoid 各 50 条，**全部 age=4**）：renjian/yunyou/shanju 的 avoid 赫然是「丙火伤官，从不信命」「红塔山，烟灰积了一截」「像一枚印章稳稳盖在西装的第二颗纽扣下方」「霓虹灯管把脸切成冷暖两半」「像三枚失效的棋子」「像掰断一根湿筷子」……全是前世/爽文残留。

## 二、各问题的系统成因（prompt / evolution / 结构 · file:line）

| 问题 | 根因位置 | 机制成因 |
|---|---|---|
| **①节奏单一** | `longrun.ts:41`(PENMANSHIP 温润) + `:179`(beatSpec) + `:203`(secPrompt「约 N 字·宁短勿堆」) | **降密度副作用联动**：M1「一段一事·宁短」(L41/L203) 压低单段信息量的**同时把详略压平**——过场和高光被同等地「写从容即收」。「疏密相间」只写了「该疏处疏」，**缺显式的速度对照指令**（没要求「敢用一句话甩一段、再用一整段驻足」），于是所有段一样轻、一样长、一样中速。**这正是用户预判的「M1 宁短催生详略不当」的联动**。 |
| **②剧本模板重演** | `longrun.ts:179`(GENTLE beatSpec「须有人来人往……一个来客」) + `:182`(换景注入「让一个新面孔自然进入」) | beatSpec/换景器**强制「换一处地点、来一个新面孔」**（防场景单一，见 memory `novel-narrative-momentum`），但只约束了「场景/面孔要变」，**没约束「来客的剧本要变」**——模型机械执行「来客」指令却复用同一种来客剧本（报丧/劝归）。`gentle-director.ts:44` motifSig 只对**名词静物指纹**去重（场景维），管不到 plot-beat 维。T2 进展账本 `progression-ledger`（防主线循环）也不记「对话事件类型」。 |
| **③意象跨章复读** | `evolve.ts:209`(amplify「以物载情/回扣」无上限) + `:263`(overused 没摘喻体骨架) + `longrun.ts:171`(dropMotifs 只剔近章静物、限于 rollSummary 纲要、且不进 avoid) | amplify 学到「豁口碗承载欲言又止」是 win（甚至明确钦点这条喻），于是**每章都端豁口出来**——`evolve.ts:357` 再现=upvote 反而强化它。`warm-fitness.ts:141` 已明文**弃用物象 lexicon**（因「一两件物反复回扣」会让物象总命中也高、反向罚回扣），所以跨章物象通胀**当前无任何信号在测**。dropMotifs(L167) 只作用于 bible 纲要压缩，不约束正文喻体、也不跨章限频。 |
| **④对白同腔** | `longrun.ts:41`(PENMANSHIP「对白重言外之意、一句别塞多条事实」单向压抑) | 单向指令「对白要短、要藏」被过度执行成「全员失声」，**没有任何「来客可絮叨、配角有声口」的反向平衡**。roster(L149-164) 注入了角色八字/羁绊/目标，但**没注入声口/语癖标签**，模型无差异化依据。evolve rubric `dialogue`(L263) 是 1-10 整体分，无反模式判据；metrics `dialogueRatio`(L238) 只测**占比**（~0.3 为佳）不测**多样性/声口**——抓不到「全员单字」。 |
| **⑤收尾套路** | `longrun.ts:203`(secPrompt 末段「换着来」是软提示) | 软提示被无视（shanju 66%）。**无任何机检统计近窗收尾类型**，自进化层 `warm-fitness.ts` 七信号无「收尾多样性」项 → 模型发现「静物渐息」安全好评（rubric `hook`=余韵回味，渐息天然得分），**越收越窄**（漂移）。 |
| **⑥avoid 空转** | `evolve.ts:80`(新世界 bootstrap 全局通用 avoid) + `:173`(全局 avoid 跨世界并集、不按 intent 隔离) + `:263`(overused 未喂温润套路) | 全局 avoid 把爽文世界的喻（西装/红塔山/霓虹）**并集后 bootstrap 进温润世界**（L80），无 `intent` 品类隔离（L173 只按跨世界频次排序）。critic 摘 `overused`(L263) 时无 GENTLE 专属指令去摘「跨章复用的喻体骨架」，于是真套路进不了 avoid。 |

**降密度 ↔ 各问题关系总述**：①是降密度的**直接副作用**（联动，须协同修，不能再加码降密度）；②④⑤是**结构/声口/收尾维**、③⑥是**跨章时间维 / 机制维**——全部落在降密度触及不到的别处。补这些**不会让文字变回「慢而满」**，反而是在已变清爽的基底上补「节奏起伏、剧本多样、声音层次、喻体新鲜」。

## 四、该【保住】的优点（别误伤）

1. **留白 / W_breath 寡淡基底**：对话/千字低 = 言外之意，是温润的命门（好章 3.7-5.9）。修 A「对白有声口差」只动**配角可絮叨**，**不要把主角的惜字也改掉**——主角仍应稀字、靠配角制造密度差。`warm-fitness.ts:141` 弃用物象 lexicon 的决策正确，别回退。
2. **以物载情 / 回扣**：「一两件物贯穿全章反复回扣」是降密度想要的、是温润质感来源。修 F/G 限的是**跨章象征通胀与喻体复用**，**不是章内回扣**——一件物在单章内反复回扣应保留，只压「连续章都端同一象征物 + 同一喻底」。
3. **章末余味本身**：好章 ch1「淡得他自己都没有察觉」是惊艳收尾。修 E/B 限的是**连续同类（66% 全静物渐息）**，不是禁用渐息——渐息仍是轮换清单里的一种，只是不能连用。
4. **温润质感 / 慢而有味**：rubric pacing=「从容张弛与留白（慢而有味为佳、不催不赶）」(evolve.ts:263) 是对的。修 A 加的是**快慢落差**，不是把节奏加快成爽文——是「慢段更慢、过场更快」的张弛，不是整体提速。
5. **密明喻**：温润允许密明喻（好章 6.09/6.64 > 在跑章），**不要去压明喻数量**。修 A/G 管的是喻体**新鲜度与象征度**，不是数量。
6. **降密度方向（2caf8ae）整体**：M1 prompt + M2 采样不回退、不加码。本报告全部补在正交维。
