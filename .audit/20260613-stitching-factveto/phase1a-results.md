# Phase 1a 结果: 确定性 veto 的离线精度 + 覆盖面天花板
> 2026-06-13 · 先离线证精度(护栏d)· 72 章判官标注语料(R 实验 4 臂: 开篇 32 + 中段 40)· 零生成 token

## 0. 一句话
确定性高精度 veto **机制成立、精度过门(0 误检/72)**,但**安全可否决的覆盖面很窄**: 仅"称谓法名漂移"干净可用; 性别 0-recall(本语料); **数值在护栏 a 安全约束下 0 产出**(真例全落在排除区)。**deterministic veto 天花板低·治不了 factC 主体。**

## 1. 三类实测(72 章·对照 judge ground-truth)
| 类 | 命中 | 误检 | 精度 | 召回(本语料) | 判定 |
|---|---|---|---|---|---|
| **称谓法名漂移** | 1(ch332 静檀↔静安师太) | 0 | **100%** | 1/1 真例 | ✅ 干净可用(限宗教称谓+法名似真) |
| **性别** 他↔她 | 0 | 0 | —(0 fire) | 0/1(林思齐"是X。她"松绑定漏) | ⚪ 安全但本语料 0-recall(紧绑定保精度) |
| **数值** | 0 | 0 | —(0 fire) | 0/多(船资5↔3/凝皮7↔9/绳长全被排) | 🔴 **护栏a安全约束→0产出** |

## 2. 两轮精度迭代(护栏d 实证有效)
- 初版误检: ①autoNames 把物象"雨沫子"当人(无姓氏锚)→性别假绑 ②"老板娘/姑娘/先生/大夫"世俗称谓 `[CJK]{2,3}称谓` 抓动词短语("说话快""娘正要")=噪声爆炸(ch334 单章 90+ 假漂移) ③"后静观"贪婪抓方位字。
- 修: 姓氏锚滤物象 + 称谓限宗教(法名似真过滤) + 名漂移跳子串 + 法名 2 字停字表。
- **结果: 90+ 假漂移 → 0 误检。先离线卡死精度挡住了雨沫子误检触发的无谓重生(护栏d 实证: 若直接接重生早烧 token+改坏好文)。**

## 3. 关键发现: 护栏 a 与数值可否决性的内禀冲突
护栏 a("数值否决须排可消耗计数/时辰/跨交易")是对的(防误检)——**但散文里真数值矛盾几乎总伴交易/变化语境**(船资"付了五文"…"船资三文"·凝皮"摞了七八片"…"又…九片")→ 排除触发一开,真例全被排。**安全=0产出·有产出=不安全。** 数值类在确定性框架下**无安全甜点**。与 lint-seams D20"容器材质漂移零误杀做不到·需指代消解=LLM"同构。

## 4. 推论: deterministic veto 天花板低
factC 体量分布(R+P2): 数值~35% + 语义(关系/来源)~30% + 称谓~25% + 性别~10% + 地点~15%(注: 地点归 transitionGap 不在此)。**确定性安全可否决的只有称谓法名漂移(~25% 里的法名子集·实际更小)+ 性别松绑定漏**。**主体(数值+语义 65%)确定性框架内治不了。**

## 5. 处置选项(交人签)
- **(a) 落地小而安全的**: 称谓法名漂移→确定性替换/重生(干净)·性别松绑定(略放宽接住"是X。她"·测精度不破)·数值/语义→只 flag 交 edit-pass(不 veto)。诚实但对 factC 主体影响小。
- **(b) Phase 2 = LLM 矛盾检查(对数据账·非散文)**: 段后用 light-LLM 拿"本段断言"对"已立数据账(entity→value)"判矛盾(喂数据非散文·避 induction)→ 这是唯一能碰数值/语义主体的路·但重引 LLM 成本+需自己的精度验证(judge 警告同模型自判虚高·或用 ≠写手模型)。架构评审与本实测都指向: 主体非确定性能解。
- **(c) 接受天花板**: deterministic veto 仅作 name-drift 小补丁·factC 主体留给章后 edit-pass(已有)兜底·不强求生成端预防。

## 6. 诚实结论
"先离线证精度"省下了真金白银(挡住雨沫子误检重生)·也**省下了一条歧路**: 不证精度就接重生+A/B·会在数值类上要么 0 产出(白跑)要么误检催肥(改坏好文)。**确定性 veto 是真但小的工具; factC 主体(数值/语义)要动须上 Phase 2 的 LLM-对数据账-矛盾检查·那是另一个 measure→build→证精度 循环。**
